# Cubist Souls — compact traits index

`index.json` is a compact, client-side trait dataset for the 10,000 Cubist
Souls. It powers the "Filter by trait" panel on the gallery (The Freed) without
any per-token metadata fetches.

Generated by [`scripts/gen_traits_index.py`](../scripts/gen_traits_index.py)
from the root `meta-all.json`. QA'd by
[`scripts/qa_traits_index.py`](../scripts/qa_traits_index.py).

Served raw at:
`https://raw.githubusercontent.com/adriangallery/cubist-souls-assets/main/traits/index.json`

## Size

~80 KB uncompressed (well under the 500 KB budget), ~15 KB gzipped.

## Shape

```json
{
  "version": 1,
  "generated": "2026-07-19",
  "count": 10000,
  "base": 48,
  "categories": ["Art Background","Base","Clothes","Mouth","Head","Left Eye","Nose","Right Eye"],
  "values":  { "Art Background": ["Alpha", "Beta", ..., "1 of 1"], ... },
  "tokens":  { "Art Background": "<10000-char string>", ... }
}
```

- **categories** — the 8 trait categories, fixed order.
- **values[cat]** — the distinct values for that category, sorted
  alphabetically, with a synthetic `"1 of 1"` appended last.
- **tokens[cat]** — a string of exactly 10,000 characters. The character at
  position `i` encodes the value index of **tokenId `i + 1`** as
  `chr(base + index)` (`base = 48`, i.e. index 0 → `'0'`). Every category value
  count is ≤ 21, so a single printable char per token suffices.

## Decoding (JS)

```js
const data = await (await fetch(URL)).json();
const B = data.base; // 48

function valueOf(category, tokenId) {
  const code = data.tokens[category].charCodeAt(tokenId - 1);
  return data.values[category][code - B];
}

valueOf("Art Background", 1); // "Star Blue"
```

## Honorary 1/1 tokens

Tokens **#90, #163, #294, #600** are honorary 1/1s with no standard traits.
They decode to `"1 of 1"` in every category. That value is the last entry of
each category's `values` list, so a "1 of 1" filter option surfaces them and
mostly reads as `0` among the freed until an honorary is actually freed.

## Regenerating

```bash
python3 scripts/gen_traits_index.py   # writes traits/index.json
python3 scripts/qa_traits_index.py     # validates against meta-all.json
```

Re-run whenever `meta-all.json` changes.
